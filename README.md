# Bubble Analysis Project

This project analyses the relationship between macro-financial indicators and S&P 500 performance during:

- Dot-com Bubble
- Housing Bubble
- Current Market

## Indicators Used

- VIX
- Chicago Fed NFCI Leverage Subindex
- SLOOS Bank Tightening Standards
- S&P 500
